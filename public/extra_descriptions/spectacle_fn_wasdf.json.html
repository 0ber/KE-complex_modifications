When using Spectacle on a Macbook, hitting fn and WASD of F will move the current window.

Fast window movement with Spectacle even when overriding modifiers in other apps.

  • fn-w: snap current window up
  • fn-a: snap current window left
  • fn-s: snap current window down
  • fn-d: snap current window right
  • fn-f: make current window fullscreen
